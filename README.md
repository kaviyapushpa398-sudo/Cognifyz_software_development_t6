# 🌐 Interactive Web Scraper (Python)

## 📌 Overview

This project is an **Interactive Web Scraping Tool** built using Python. It allows users to fetch and display useful data such as quotes, headlines, or product details from websites in a clean and structured format.

---

## 🎯 Objective

To develop a user-friendly application that extracts data from websites and presents it in a readable and interactive way.

---

## 🏢 Project Source

This project was completed as part of a task provided by **Cognifyz**.

---

## 🛠️ Features Implemented

### ✅ 1. Dynamic Website Input

* Allows users to input a website URL
* Supports scraping from simple, public websites

---

### ✅ 2. Data Extraction

* Uses Python libraries:

  * `requests`
  * `BeautifulSoup`
* Extracts data such as:

  * Quotes
  * Headlines
  * Product information

---

### ✅ 3. Interactive Menu

* Menu-driven interface:

  1. Scrape Quotes
  2. Scrape News Headlines
  3. Scrape Product Data
  4. Exit

---

### ✅ 4. Clean Output Display

* Displays scraped data in a structured format
* Numbered lists for better readability

---

### ✅ 5. Error Handling

* Handles:

  * Invalid URLs
  * Network issues
  * Missing HTML elements
* Displays user-friendly error messages

---

### ✅ 6. Multiple Operations

* Allows users to perform multiple scraping tasks in one run
* Option to continue or exit

---

### ✅ 7. Bonus Features

* Save scraped data to a file (TXT/CSV)
* Limit number of results (Top 5 / Top 10)
* Add timestamps to scraped data

---
